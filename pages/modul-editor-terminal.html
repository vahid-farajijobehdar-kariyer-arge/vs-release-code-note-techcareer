Kategori Dagilimi

Kitty Graphics Protocol

Terminalde yuksek kaliteli gorsel rendering icin yeni protokol. Etkinlestirmek icin:

{ "terminal.integrated.enableImages": true, "terminal.integrated.gpuAcceleration": "on" }

Desteklenen Formatlar

  • PNG
  • 24-bit RGB
  • 32-bit RGBA
  • Chunked base64 + zlib sikistirma

Ozellikler

  • Olcekleme ve kirpma
  • Piksel ofsetleri
  • Z-index katmanlamasi
  • Gorsel yonetimi: depola, yerlesim, sil, guncelle
Not: Animasyonlar, goreli yerlesimler, Unicode yer tutuculari ve dosya tabanli aktarim henuz desteklenmemektedir.
AyarAciklama
terminal.integrated.enableImagestrue olarak ayarla
terminal.integrated.gpuAccelerationon veya auto olmali
terminal.integrated.windowsUseConptyDllWindows icin gerekli

JS/TS Ayar Birlestirme

Tum JS/TS ayarlari tek js/ts.* prefiks altinda toplandi. Eski ayarlar calismaya devam eder ama deprecated sayilir.

// Oncesi (eski - kullanim disi) "javascript.format.enable": true, "typescript.format.enable": true // Sonrasi (yeni ve birlesik) "js/ts.format.enabled": true // Dile ozgu farklilik icin "[javascript][javascriptreact]": { "js/ts.format.enabled": false }, "[typescript][typescriptreact]": { "js/ts.format.enabled": true }

Next Edit Suggestions (NES)

Artik dosyanin herhangi bir yerini tahmin edebilir — sadece imleq yakinligi degil.

AyarAciklama
github.copilot.nextEditSuggestions.enabledNES'i etkinlestir
github.copilot.nextEditSuggestions.extendedRangeUzak mesafe modu
Pratik ornek: Bir fonksiyon adini degistirdiginde NES, dosyanin basindaki JSDoc yorumunu da guncellemenizi onerir.

Copilot Status Bar'da NES isteklilik seviyesi ayarlanabilir: daha fazla oneri (daha az alakali) veya daha az oneri (daha alakali).

Modal Editor (Deneysel)

Sekme duzenini etkilemeden uzerine kayan yeni editor tipi.

{ "workbench.editor.useModal": "some" }

Gecerli oldugu yerler: Ayarlar, Klavye Kisayollari, Profiller, AI Modelleri, Calisma Alani Guveni.

Bildirim Konumu

{ "workbench.notifications.position": "top-right" }

Secenekler: top-right, bottom-right (varsayilan), bottom-left
